# Acoplamento estrutural

Acoplamento estrutural é o que permite que um sistema seja uma rede coordenada, mesmo que cada parte dele (Direito, Religião, Ciência, no caso de sistemas sociais) siga apenas suas próprias regras.

Um [[20260307105528|sistema]] não recebe ordens de fora, pois um não pode dizer ao outro o que fazer. Porém, um _perturba_ (ou _irrita_) o outro. Acoplamento estrutural é quando essa perturbação se torna permanente e canalizada. Os sistemas passam a contar com a existência um do outro para funcionar.

O sistema precisa do ambiente para ter o que processar. Porém, traduz tudo internamente para sua própria linguagem.
