# Programa

São as condições, critérios estruturais e conteúdos que orientam o sistema a aplicar o [[20260413105744|código binário]] em uma situação concreta. O [[20260413105744|código]] define o que o sistema faz, enquanto o programa define como ele decide fazer.

Os códigos permanecem na parte [[20260306181359|fechada]] do sistema sem influência direta do ambiente. Os programas funcionam como uma antena do sistema para o mundo exterior. Permitem aos subsistemas realizar aberturas ao ambiente para adquirir capacidade de aprender, transformar-se e reproduzir suas estruturas ao longo do tempo. Ao contrário dos códigos, os programas são mutáveis e [[20260309214436|contingentes]].

O [[20260323094013|ambiente]] externo é caótico e hipercomplexo. Os programas filtram essa complexidade, traduzindo as variações do ambiente em um formato compreensível para o código do sistema.

Por exemplo, o juiz utiliza as normas legais (o programa) para reduzir a [[20260309221929|complexidade]] e emitir um veredito final focado estritamente na determinação da culpa (o [[20260413105744|código]] lícito/ilícito).

## Exemplos

| Sistema | Programa | Código |
| :--- | :--- | :--- |
| **Direito** | Normas legais | Balizam o código lícito/ilícito |
| **Economia** | Preços | Sobem e descem orientando decisões de pagamento/não-pagamento |
| **Ciência** | Teorias e os experimentos científicos | Orienta se uma pesquisa atinge o valor verdadeiro |
| **Política** | Programas eleitorais, programas de governo, políticas públicas | Governo/oposição | 
| **Meios de massa** | Notícias e reportagens, publicidade, entretenimento | Informativo/não informativo |
