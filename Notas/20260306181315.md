# Autopoiese

Autopoiese é a capacidade que um [[20260307105528|sistema social]] tem de produzir e reproduzir seus elementos a partir de seus próprios elementos.

Forma pela qual a comunicação se organiza em sistemas autônomos (Direito, Economia, Ciência), que se mantêm vivos independentemente das intenções individuais dos seres humanos. 

Os elementos do sistema (comunicações ou pensamentos) são eventos efêmeros e desaparecem no instante em que ocorrem. O sistema é forçado a produzir uma nova operação o tempo todo para não deixar de existir. A produção desses elementos é, portanto, uma corrida constante contra o fim do sistema.
