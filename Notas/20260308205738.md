# Simultaneidade

Operação única que aponta para duas direções opostas no exato momento em que acontece.

Toda e qualquer comunicação possui duas faces que ocorrem no mesmo instante: 

- [[20260308200131|Autorreferência]]
- [[20260308201523|Heterorreferência]]

Não pode haver uma sem a outra. Se houver apenas autorreferência, o sistema é autismo operacional, pois não fala de nada. Se houver apenas heterorreferência, o sistema se dissolve e perde sua identidade. A simultaneidade é o que mantém o sistema operando como uma unidade funcional.

O [[20260306181339|sistema]] faz uma distinção entre si e o entorno, que é realimentada dentro do próprio sistema. Ele sabe simultaneamente que é um sistema e também que existe um mundo fora.
