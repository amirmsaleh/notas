# Alopoiese

Alopoiese é o processo pelo qual um sistema produz um resultado externo a si mesmo, em vez de [[20260306181315|reproduzir]] seus próprios elementos. Ou seja, o produto do sistema não é o próprio sistema. Isso é o oposto à definição de [[20260307105528|sistema]] [[20260306181315|autopoiético]].

Características principais:

- Produção externa: o sistema gera algo diferente de sua própria estrutura;
- Dependência de finalidade externa: o sistema normalmente existe para produzir algo para fora;
- Reprodução não interna: a operação do sistema não gera automaticamente os elementos que o constituem.

Tipos de sistemas:

- Biológicos: [[20260306181315|autopoiéticos]];
- Sociais: [[20260306181315|autopoiéticos]], pois produzem comunicação;
- Máquinas e software: geralmente alopoiéticas.
