# Autorreferência basal

Autorreferência basal é definida como o nível mais elementar de fechamento operacional. É a referência direta de um elemento a outro elemento do próprio sistema.

A autorreferência tem três níveis de complexidade ascendente:

1. Autorreferência Basal: A relação pura e direta entre elementos mínimos do sistema.

2. Reflexividade: Referência no nível dos processos (uma operação que regula outra operação, como a comunicação sobre a comunicação ou uma seleção sobre outra seleção.

3. Reflexão ou [[20260329172810|reentrada]]: A referência no nível do sistema total, onde o sistema observa a si mesmo como uma unidade em contraposição ao seu ambiente.

Os elementos constituintes dos sistemas [[20260306181315|autopoiéticos]]  são acontecimentos instantâneos que desaparecem no exato momento em que emergem. Como os elementos não têm permanência física ou substancial, a [[20260306181315|autopoiese]] seria impossível se cada acontecimento operasse de forma isolada. Para que o sistema não colapse a cada instante um elemento só pode adquirir unidade, sentido e existência se ele se ligar recursivamente a um elemento anterior do mesmo sistema e, simultaneamente, abrir o horizonte de possibilidades para a conexão de um elemento posterior.

A autorreferência basal é o motor da reprodução autopoiética: a produção de elementos por meio de elementos. O sistema não extrai a sua unidade de uma substância externa ou de um sujeito, mas sim da rede contínua de remissões internas entre os seus próprios acontecimentos.

No [[20260307105528|sistema social]] o único elemento componente é a comunicação. A autorreferência basal se manifesta na ligação recursiva entre eventos comunicativos. Uma unidade de [[20260309221624|comunicação]] faz referência a uma comunicação que a precedeu e projeta uma expectativa para uma comunicação que a sucederá.

Exemplo: em um diálogo a emissão de uma frase só adquire o status de elemento do sistema social se for compreendida em referência ao que foi dito antes. É essa ligação basal que gera a estrutura elementar do sentido e viabiliza a emergência e a manutenção de qualquer ordem social.
