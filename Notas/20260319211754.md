# Autorreferência basal

É o motor da reprodução [[20260306181315|autopoiética]]: a produção de elementos por meio de elementos. O produz sua unidade a partir de uma contínua de remissões internas entre os seus próprios acontecimentos. É o nível mais elementar de [[20260306181359|fechamento operacional]], onde há referência direta de um elemento a outro dentro do próprio sistema.

A autorreferência tem três níveis de complexidade ascendente:

1. Autorreferência Basal: A relação pura e direta entre elementos mínimos do sistema.

2. Reflexividade: Referência no nível dos processos (uma operação que regula outra operação, como a comunicação sobre a comunicação ou uma seleção sobre outra seleção.

3. Reflexão ou [[20260329172810|reentrada]]: A referência no nível do sistema total, onde o sistema observa a si como uma unidade em contraposição ao seu ambiente.

Os elementos constituintes dos sistemas [[20260306181315|autopoiéticos]]  são acontecimentos instantâneos que desaparecem no exato momento em que emergem. Como os elementos não têm permanência física ou substancial, a [[20260306181315|autopoiese]] seria impossível se cada acontecimento operasse de forma isolada. Para que o sistema não colapse a cada instante um elemento só pode adquirir unidade, sentido e existência se hovuer ligação recursiva a um elemento anterior e se houver a abertura de um horizonte de possibilidades para a conexão de um elemento posterior.

No [[20260307105528|sistema social]] o único elemento componente é a comunicação. A autorreferência basal se manifesta na ligação recursiva entre eventos comunicativos. Uma unidade de [[20260309221624|comunicação]] faz referência a uma comunicação que a precedeu e projeta uma expectativa para uma comunicação que a sucederá.

Exemplo: em um diálogo a emissão de uma frase só adquire o status de elemento do sistema social se for compreendida em referência ao que foi dito antes.
