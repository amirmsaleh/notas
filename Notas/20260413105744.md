# Código binário

Códigos binários são a gramática fundamental dos [[20260307105528|sistemas sociais]]. Permitem a um sistema se diferenciar do seu [[20260308194134|entorno]] e processar [[20260309221624|comunicações]] de forma eficiente.

O código binário age como uma lente que só enxerga duas cores. Se o sistema não consegue traduzir um evento para essas duas cores ele o ignora.

Todo sistema funcional opera com uma distinção básica que não aceita meio-termo (terceiro excluído). Um lado é o valor de atribuição (o que o sistema quer) e o outro é o valor de reflexão (o que o sistema evita ou usa para se questionar).

| Sistema | Código Binário | Significado |
| :--- | :--- | :--- |
| **Direito** | Lícito / Ilícito | O evento viola a norma ou está de acordo com ela? |
| **Ciência** | Verdade / Falsidade | A afirmação é cientificamente válida ou não? |
| **Economia** | Ter / Não ter | Há dinheiro para realizar a transação ou não? |
| **Política** | Governo / Oposição | A decisão mantém o governo ou fortalece a oposição? |
| **Religião** | Transcendência / Imanência | O evento tem sentido divino ou é puramente mundano? |

A principal função do código binário é a redução de complexidade, pois dessa forma o sistema limpa o ruído, permitindo que o sistema tome uma decisão técnica imediata.

Código não se confunde com [[20260525110046|programa]]. Os programas dizem como o código deve ser aplicado.

Os códigos garantem que um sistema seja [[20260306181359|fechado operacionalmente]]. A Ciência não aceita um argumento "político" como prova de uma verdade científica, assim como a Economia não aceita uma "oração" como pagamento de uma dívida. Cada sistema é autônomo porque seu código é incomunicável para os outros.
