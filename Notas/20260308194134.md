# Entorno e Forma
Forma é uma fronteira que separa um espaço em um lado indicado (marcado) e um lado não indicado (não marcado).

[[20260307105528|Sistema]] é o lado indicado da distinção.

**Entorno** é o lado não indicado.

**Forma** é como uma linha que separa os dois. A forma de um sistema é a distinção entre sistema e entorno.

O sistema não tem uma fronteira: ele é a própria operação de traçar essa fronteira. Se a distinção deixa de ser feita, o sistema desaparece.
