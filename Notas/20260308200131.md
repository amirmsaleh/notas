# Autorreferência

Um sistema só existe enquanto for capaz de se referir a si mesmo para realizar sua próxima operação.
Se o sistema parasse de olhar para si mesmo, ele seria absorvido pelo ambiente e deixaria de ter uma identidade própria.

A operação atual só acontece porque se baseia em operações passadas do próprio sistema, num processo circular de [[20260306181315|autopoiese]].

É essa circularidade que dá autonomia ao sistema, pois este decide o que é válido dentro de sua própria lógica. Quando um sistema observa o mundo, não consegue ver a lógica que está usando no momento da [[20260323085754|observação]] (ponto cego), pois esta é inerente.

[[20260308200131|Autorreferência]] e heterorreferência acontecem [[20260308205738|simultaneamente]].
