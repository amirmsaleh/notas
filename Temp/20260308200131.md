# Autorreferência

Um sistema só existe enquanto for capaz de se referir a si mesmo para realizar sua próxima operação.
Se o sistema parasse de olhar para si mesmo, ele seria "engolido" pelo ambiente e deixaria de ter uma identidade própria.

A operação atual só acontece porque se baseia em operações passadas do próprio sistema, num processo circular de [[20260306181315|autopoiese]].

Essa circularidade que dá autonomia ao sistema, pois este decide o que é válido dentro de sua própria lógica.

Quando um sistema usa sua própria lógica para observar o mundo, ele não consegue ver a própria lógica que está usando no momento da observação (ponto cego).

[[20260308200131|Autorreferência]] e heterorreferência acontecem [[20260308205738|simultaneamente]].
