# Heterorreferência

É o que permite que o sistema tenha conteúdo e não seja apenas um eco vazio de si mesmo.

É a capacidade de um sistema [[20260306181315|autopoiético]] de designar algo como "não pertencente". O sistema constrói uma imagem de seu ambiente dentro de si mesmo, criando uma informação sobre o mundo.

[[20260306181339|Sistemas]] são operacionalmente [[20260306181359|fechados]], mas cognitivamente abertos. Essa abertura acontece pela heterorreferência. 

O sistema permite ser [[20260306174301|irritado]] ou "estimulado" pelo que acontece fora. 
Sem a heterorreferência o sistema não teria "assunto". O Direito, por exemplo, ficaria repetindo leis sem nunca aplicá-las a conflitos sociais reais.

[[20260308200131|Autorreferência]] e heterorreferência acontecem [[20260308205738|simultaneamente]].
