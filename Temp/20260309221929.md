# Complexidade para Luhmann
A complexidade é a medida da incerteza. Situação onde existem mais elementos do que a capacidade do sistema de conectá-los entre si.

Quando o número de elementos cresce o sistema é obrigado a selecionar quais conexões vai manter.

O mundo (entorno) é infinitamente complexo e o sistema é sempre menos complexo que seu ambiente.

Exemplos:

- O Sistema Jurídico reduz a complexidade de um conflito humano a: "isso é legal ou ilegal"?
- O Sistema Econômico reduz a complexidade das necessidades humanas a "quanto custa" ?

Paradoxo: quanto maior a complexidade de um sistema melhor é sua resposta aos estímulos do entorno. 

4. Complexidade e Tempo
A complexidade também é temporal. Como os elementos do sistema (comunicações) são efêmeros e morrem instantaneamente, o sistema vive sob a pressão de que a próxima conexão seja feita.
A complexidade gera contingência: "o que está acontecendo agora poderia ser de outro modo".
O sistema precisa escolher um caminho, descartando outros, o que gera risco.
