# Reentrada
Reentrada é o que acontece quando a distinção original [[20260307105528|sistema]]/[[20260308194134|entorno]] é copiada dentro do próprio sistema. O sistema passa a tratar essa separação como uma informação interna. No momento em que a diferença entre sistema e entorno aparece dentro do sistema ele se torna capaz de observar a si mesmo como algo diferente do mundo.

Sem a reentrada o sistema apenas reage ao mundo. Com a reentrada o sistema ganha reflexividade. 

## Exemplos 
* O sistema jurídico usa a distinção Direito / Não-Direito. A reentrada ocorre quando o Direito precisa decidir, juridicamente, se uma norma jurídica é válida. Ou seja, o Direito aplica sua própria distinção sobre si mesmo.
 * O sistema econômico opera com a distinção Pagamento / Não-pagamento. A reentrada ocorre quando o sistema começa a precificar o próprio dinheiro (juros). O dinheiro, que é o meio, torna-se o objeto da operação.
 * A ciência observa o mundo para encontrar a Verdade. A reentrada ocorre quando a ciência começa a estudar a própria ciência (epistemologia).
