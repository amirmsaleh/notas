# Sistemas autorreferenciais

## Ideia

Um sistema é autorreferencial quando usa a si mesmo como ponto de partida e de chegada para suas próprias operações. Ele é surdo a qualquer lógica que não seja a sua.

Sistemas autopoiéticos  são necessariamente autorreferenciais. 

Sistemas alopoiéticos não são. 

O objeto da teoria de sistemas deixa de ser o sistema em si para se tornar a diferença no mundo entre o sistema e o ambiente.
