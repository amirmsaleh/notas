# Fechamento operacional

## Ideia

O sistema é surdo a qualquer lógica que não seja a sua. Como o sistema usa seus próprios olhos (seus códigos) para ver o mundo, ele não consegue ver o que está fora. É prisioneiro de sua própria lógica autorreferencial, pois não podem ser diferentes do que o seu código permite.
